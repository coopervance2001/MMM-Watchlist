# MMM-Watchlist
[MagicMirror²](https://modules.magicmirror.builders/) module that displays a stock/ETF watchlist with:
- real-time-ish quotes
- pre-market + after-hours columns
- sparklines (1D chart)
- caching + automatic rate-limit backoff

## Installation
```bash
cd ~/MagicMirror/modules
git clone https://github.com/coopervance2001/MMM-Watchlist
cd MMM-Watchlist
npm install
